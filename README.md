## 📱 Technical Architecture

### 🛠️ Core Technology Stack
- **Frontend**: HTML5, CSS3 Grid/Flexbox, Vanilla JavaScript ES6+
- **Voice AI**: Web Speech API (Recognition + Synthesis)
- **Audio**: Web Audio API for dynamic coin sound generation
- **Data**: localStorage for persistence, sessionStorage for temporary data
- **Export**: Blob API for multi-format downloads (CSV, JSON, PDF-ready)
